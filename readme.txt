== Description ==

Responsive Flexible Slide To Top Accordion is a free Wordpress Accordion Plugin. It's a good choice to display additional information with accordion effect. As an administrator, you can set number of items to be displayed.Responsive Flexible Slide To Top Accordion is built with flexible features: responsive layout, unlimited accordions content, unlimited galleries.
